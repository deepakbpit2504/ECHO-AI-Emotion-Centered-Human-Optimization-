# 🌿 ECHO AI - Emotion-Centered Human Optimization

ECHO AI is a Streamlit-based AI web application that analyzes human emotions using Natural Language Processing and provides personalized insights to improve mental well-being and productivity.

---

## 🚀 Features

- 🔐 User Registration & Login system
- 🧠 Emotion detection using NLP (TextBlob)
- 📊 Interactive dashboard for emotion analysis
- 📈 Emotion history tracking
- 📉 Visualization of emotional trends
- 💡 Rule-based recommendation engine
- 🗄️ SQLite database for persistent storage
- 📄 Multi-page Streamlit architecture

---

## 🏗️ Tech Stack

- Python
- Streamlit
- TextBlob (NLP)
- SQLite
- Pandas
